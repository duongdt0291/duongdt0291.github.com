JS Concept: Object (P3)

17 tháng 1 2021

Property Descriptor

Các thuộc tính (property) trong 1 object không chỉ đơn thuần chứa các giá trị mà còn có các tính chất khác (descriptor). Để xem các tính chất này, chúng ta dùng Object.getOwnPropertyDescriptor(obj) .

const user = {
  name: 'duongdt',
}

console.log(Object.getOwnPropertyDescriptor(user, 'name'))
// {
//		value: 'duongdt',
//		writable: true,
//    enumerable: true,
//    configurable: true
// }

Các tính chất trên mang ý nghĩa sau:

  • value : giá trị của thuộc tính
  • writable : có thể chỉnh sửa giá trị của thuộc tính được không.
  • configurable : có thể chỉnh sửa các property descriptor được hay không. Ngoài ra nếu là false, thì sẽ không thể xóa thuộc tính (writable = false chỉ giúp không chỉnh sửa được giá trị của thuộc tính nhưng vẫn có thể xóa được).
  • enumerable: có thể dùng vòng lặp để lấy ra thuộc tính hay không.

Để tạo 1 thuộc tính bình thường, cách viết thông thường sẽ là user.age = 30 , lúc này các descriptor của trường age sẽ lấy giá trị mặc định là true.

Nếu muốn cài đặt hoặc chỉnh sửa các tính chất này, chúng ta dùng Object.defineProperty .

const user = {
  name: 'duongdt',
  age: 30,
  job: 'developer',
}

// writable
user.name = 'dima'
console.log(user.name) // dima
Object.defineProperty(user, 'name', {
  writable: false,
})
user.name = 'duongdt'
console.log(user.name) // dima

// enumerable
console.log(Object.keys(user)) // [ 'name', 'age', 'job' ]
Object.defineProperty(user, 'name', {
  enumerable: false,
})
console.log(Object.keys(user)) // [ 'age', 'job' ]
console.log('name' in user) // true

// configurable
Object.defineProperty(user, 'name', {
  configurable: false,
})
delete user.name
console.log(user.name) // 'dima'
console.log(user.age) // 30
delete user.age
console.log(user.age) // undefined

Getters, setters

Đôi khi khi thay đổi giá trị của 1 thuộc tính chúng ta cần validate hoặc 1 thuộc tính có giá trị bằng 2 hoặc nhiều thuộc tính khác hợp lại. JS cung cấp getter và setter để hỗ trợ chúng ta việc này.

const user = {
  first_name: 'Duong',
  last_name: 'DT',
  get full_name() {
    return `${this.first_name} ${this.last_name}`
  },
  set full_name(name) {
    const els = name.split(' ')
    this.first_name = els[0]
    this.last_name = els.slice(1).join(' ')
  },
}
console.log(user.full_name)
user.full_name = 'Duong Dang Tuan'
console.log(user.last_name)

Immutability

JS cung cấp 1 số build-in function để tạo ra tính bất biến cho object.

  • Object.preventExtensions(obj): không cho phép thêm thuộc tính mới.
  • Object.seal(obj): không cho thêm, xóa thuộc tính. Đặt configurable = false cho tất cả các thuộc tính.
  • Object.freeze(obj): không cho thêm, sửa, xóa thuộc tính. Đặt configurable = false, writable = false cho tất cả thuộc tính.
