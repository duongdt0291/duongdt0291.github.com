JS Concept: This (P2)

10 tháng 1 2021

Từ khóa this là 1 trong những vấn đề dễ gây đau đầu và nhầm lẫn nhất cho những người mới tiếp xúc với JS. Vậy this là quái gì?

this là 1 biến chứa đường dẫn đến 1 object được gán khi 1 function được gọi, và chỉ được xác định dựa vào ngữ cảnh (context) khi function đó được gọi.

Để xác định được this chỉ cần nhớ được các qui tắc sau.

  • Global binding: Khi gọi 1 function ở ngữ cảnh toàn cục, thì this của function đó sẽ là biến global (tùy môi trường có tên gọi khác nhau, window ở browser, global ở nodejs, …) ở chế độ non-strict mode, hoặc undefined ở chế độ strict mode
let a = 10

function foo() {
  console.log(this.a)
}

foo() // 10 - ở chế độ non-strict
  • Implicit binding
function foo() {
  console.log(this.name)
}

const obj = {
  name: 'Duongdt',
  foo,
}

obj.foo() // Duongd
  • Explicit binding: Chúng ta cũng có thể khai báo trực tiếp giá trị cho this khi gọi 1 function bằng cách sử dụng bộ ba Apply, Call, Bind . Đây là 3 method thuộc về Function.prototype. Cú pháp để sử dụng như sau:
function foo() {
  console.log(this.name)
}

const obj = {
  name: 'Duongdt',
  foo,
}

const obj2 = {
  name: 'mrhayxinhun',
}

foo.call(obj2) // mrhayxinhun
foo.apply(obj2) // mrhayxinhun
obj1.foo.call(obj2) //mrhayxinhun
const fooBindedByObj2 = foo.bind(obj2)
fooBindedByObj2() //mrhayxinhun
fooBindedByObj2.call(obj) //mrhayxinhun
  • New binding
function Car(name, color) {
  /*
   * Khi gọi 1 function với từ khóa new,
   * JS sẽ tạo ngầm 1 object và gán cho biến this ở đầu function
   * Sau khi chạy xong toàn bộ code trong function sẽ trả về biến đó
   */
  // const this = {}
  this.name = name
  this.color = color
  // return this

  // **Lưu ý**: Nếu function có return 1 giá trị khác, thì giá trị đó sẽ được trả về chứ không phải this
}

const rangeRover = new Car('Range Rover', 'black')
  • Arrow Function Arrow function không có this , hay có thể hiểu là EC (execution context) được tạo khi arrow function được gọi ko xác định giá trị của this , thay vào đó nó lấy EC gần nhất có this để lấy this đó về làm this của mình.
const user = {
  name: 'Duong',
  languages: ['JavaScript', 'Ruby', 'Python'],
  greet() {
    const hello = `Hello, my name is ${this.name} and I know`

    const langs = this.languages.reduce(function (str, lang, i) {
      if (i === this.languages.length - 1) {
        return `${str} and ${lang}.`
      }

      return `${str} ${lang},`
    }, '')

    console.log(hello + langs)
  },
}
user.greet() // Cannot read property 'length' of undefined
// Nguyên nhân lỗi ở đây là callback của hàm reduce không nhận this là biến user mà sẽ là global obj ở nonstrict mode hoặc undefined ở strict mode.

const user = {
  name: 'Duong',
  languages: ['JavaScript', 'Ruby', 'Python'],
  greet() {
    const hello = `Hello, my name is ${this.name} and I know`

    const langs = this.languages.reduce((str, lang, i) => {
      if (i === this.languages.length - 1) {
        return `${str} and ${lang}.`
      }

      return `${str} ${lang},`
    }, '')

    console.log(hello + langs)
  },
}
user.greet() // Hello, my name is Duong and I know JavaScript, Ruby, and Python.
// Ở đây, chúng ta dùng arrow function, do ko tự có biến this,
// do vậy, nó trỏ lên EC ngoài của nó để tìm this,
// mà gần nhất là EC của method greet với this là obj user
